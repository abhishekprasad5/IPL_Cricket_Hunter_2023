# IPL 2023 Cricket Player Hunter 

This project is designed to showcase the team players of all the participating teams in the Indian Premier League (IPL) 2023.

## Project Overview
The IPL 2023 Cricket Player Hunter Webpage is a web application that provides an interactive and user-friendly platform to explore and discover the players of each team in the IPL 2023. Whether you're a cricket enthusiast, or just curious about the lineup of your favorite team, this webpage has got you covered.

## Features
Team-wise Player Lists: Easily navigate through the teams and view the list of players for each squad.
Responsive Design: The webpage is designed to be accessible and user-friendly across various devices, ensuring a seamless experience on desktops, tablets, and smartphones.

## How to Use
1. Clone the repository:
   ```bash
    https://github.com/abhishekprasad5/IPL_Cricket_Hunter_2023.git
    ```
2. Open the `index.html` file in your preferred web browser to start exploring the IPL player lists.
